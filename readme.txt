== Description ==

Selection du mode normal ou heroic.Possibilite de mettre le temps du mode defis.

Contiens les raids Tier 10 jusqu&#146;au tier 15. 

Ajout de la section WOTLK 

Wow progress FR peux etre configure dans le menu widget 

Themes et backgrounds dans les options > WoW Progress menu.

en cas de bug contactez moi a mstuning@free.fr

== Installation ==

1. Uploadez le plugin dans votre repertoire `/wp-content/plugins/` 
2. Activez le plugin dans le menu plugin de wordpress
3. Le widget se place simplement sur le cote

== Frequently asked questions ==

= How do i change the look of plugin? =

You can create new plugin theme by creating new <theme>.css file in `/wp-content/plugins/wowprogress/themes/` folder. Then select this theme from Settings > WoW Progress.

= How do i change background images of raids? =

You can either replace image of raid in `/images/raids/` folder, or you can upload your own and change filename in raids.json.
